### 실행 방법
```
cd backend
.venv\Scripts\activate          # Windows
source .venv/bin/activate       # macOS/Linux
uvicorn main:app --reload --port 8000
```

### curl 테스트 (Phase 1 완료 확인용)
```bash
# 분석
curl -X POST http://localhost:8000/api/analyze -F "image_a=@사진.jpg"

# 보정
curl -X POST http://localhost:8000/api/correct \
  -F "image_a=@레퍼런스.jpg" \
  -F "image_b=@대상.jpg" \
  -F "strength=0.75" \
  --output 결과.jpg
```
